Kalavidara-Balaga

Kalavidara-Balaga is an Android application developed to connect traditional folk artists with event organizers and the general public through a digital platform. The application helps rural artist troupes showcase their talent by creating profiles containing details such as art form, district, performance images, equipment information, and contact details.

Users can browse artist profiles, search and filter artists based on district and art type, and directly contact artists through the integrated call feature. The project is developed using Android Studio, Kotlin, XML layouts, and Firebase Firestore, with support from Generative AI tools for workflow planning and application structuring.

The project follows a structured Android application architecture with organized file management for better scalability and maintainability. The application contains separate modules and files for activities, adapters, data handling, layouts, themes, and Firebase integration. Key files such as `MainActivity.kt`, `DetailActivity.kt`, `ArtistAdapter.kt`, and `ArtistData.kt` manage the core functionality of the application, while XML layout files such as `activity_main.xml`, `activity_detail.xml`, and `item_artist.xml` handle the user interface design and screen layouts.

The file structure is organized to improve readability, maintainability, and future development. Resource files including colors, themes, and string resources are separated properly inside the `res` directory, following standard Android development practices. The project also includes Gradle configuration files and Firebase setup files required for dependency management and cloud integration.

The application currently demonstrates core functionalities such as artist listing, profile viewing, image loading, search and filtering concepts, and direct communication through phone call intents. Images are loaded dynamically, and artist details are displayed in a structured and user-friendly format to provide a smooth browsing experience.

Firebase integration is included to support real-time data storage and future scalability. The application architecture is designed in a way that allows future enhancements such as authentication systems, online booking features, event scheduling, ratings and reviews, multilingual support, and AI-generated artist descriptions.

The project also focuses on responsive UI design and lightweight performance so that the application can work efficiently even on devices with limited resources or slower internet connectivity. The interface uses traditional design inspirations and vibrant color combinations to reflect Karnataka’s cultural identity and folk traditions.

Kalavidara-Balaga is not only a technical Android application project but also a socially impactful platform aimed at preserving traditional art forms, improving employment opportunities for rural artists, and digitally connecting rural cultural talent with urban audiences and event organizers through modern mobile technology.
